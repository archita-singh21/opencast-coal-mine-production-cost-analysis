# ⛏️ Production and Cost Analysis of an Opencast Coal Mine

An Excel-based mine planning model developed to analyze the production capacity, operating cost, revenue, and profitability of an opencast coal mine using shovel–dumper operational parameters. The project demonstrates how changes in equipment deployment and operating conditions influence mine productivity and financial performance.

---

## Project Overview

This project simulates the production cycle of an opencast coal mine by incorporating key mining and economic parameters such as:

- Working days and working hours
- Number of dumpers
- Dumper capacity
- Dumper cycle time
- Shovel efficiency
- Coal selling price
- Operating costs

The model automatically calculates production, operating cost, annual revenue, and profit while providing graphical visualization of important mining parameters.

---

## Objectives

- Estimate annual coal production.
- Evaluate operating costs and profitability.
- Analyze the impact of dumper fleet size on production.
- Study major operating cost components.
- Demonstrate the application of Microsoft Excel in mine planning and decision-making.

---

## Tools Used

- Microsoft Excel
- Charts & Data Visualization
- Engineering Calculations

---

## Input Parameters

- Working days per year
- Working hours per day
- Number of dumpers
- Dumper capacity
- Dumper cycle time
- Shovel efficiency
- Coal selling price
- Diesel cost
- Labour cost
- Maintenance cost
- Blasting cost

---

## Key Outputs

The model calculates:

- Daily coal production
- Annual coal production
- Total operating cost
- Annual revenue
- Annual profit

It also includes visualizations showing:

- Effect of dumper fleet size on production
- Daily operating cost distribution

---

## Key Findings

- Production increases with the number of dumpers.
- Dumper cycle time has a significant impact on mine productivity.
- Diesel and maintenance contribute the largest share of operating costs.
- Efficient shovel–dumper matching minimizes idle time and improves production efficiency.
